# CrochetBloom

A responsive e-commerce website for handmade crochet products.

## Project Status
 Currently under development. Implementing new features . 

## Project Overview

CrochetBloom is a front-end e-commerce website designed to showcase and sell handmade crochet products such as bouquets, flowers, keychains, plushies, and home décor items.

## Planned Features

- Responsive Design
- Product Catalog
- Product Categories
- Product Search
- Shopping Cart
- Product Detail Pages
- Contact Form
- Local Storage Support

## Technologies Used

- HTML5
- CSS3
- JavaScript
